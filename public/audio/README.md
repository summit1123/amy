# 음성 파일 설명 - 완전한 대화 시뮬레이션

음성 파일들을 이 폴더에 넣어주세요:

## 필요한 파일들:

### AI 음성 파일:
1. **step1.mp3** - 상황 제시 (자동 재생)
   - "신나는 축구 시간!"
   - "어? 저런! 친구가 공을 뺏어갔어요. 정말 속상한 기분이에요."
   - "이럴 땐 어떻게 하면 좋을까? 마이크에 대고 네 생각을 들려줘."

2. **step2.mp3** - AI 공감 + 격려
   - "그랬구나... 친구가 공을 뺏어가서 정말 많이 속상했구나. 아미도 그 마음 알아."
   - "하지만, 친구랑 같이 놀면 더 재미있을 거야. 용기를 내서, 친구에게 '우리 같이 놀자!' 하고 말해보는 건 어떨까? 아미가 옆에서 응원할게!"

3. **step3.mp3** - AI 축하 + 보상
   - "우와! 해냈구나! '같이 놀자!'라고 말했더니, 친구가 활짝 웃으며 다시 함께 놀고 있어!"
   - "미션 성공! 용기을 낸 민준이에게 '반짝이는 축구 유니폼' 아이템을 선물로 줄게!"

### 아이 음성 파일:
4. **child_step1.mp3** - 아이 첫 반응 (회피)
   - "몰라! 흥! 그냥 나 혼자 할래!"

5. **child_step2.mp3** - 아이 두 번째 반응 (용기)
   - "...같이... 같이 놀자!"

## 완전한 시연 플로우:
1. **페이지 진입** → step1.mp3 자동 재생 (상황 제시)
2. **첫 마이크 클릭** → child_step1.mp3 → step2.mp3 → 이미지 전환
3. **두 번째 마이크 클릭** → child_step2.mp3 → step3.mp3 → 성공 팝업

완전한 대화형 AI 시뮬레이션이 구현됩니다!
